# ClipSorter Installation & User Guide

Welcome! This guide will help you download, install, and run **ClipSorter** on your computer. We've designed this to be very simple, even if you aren't a "tech person."

---

## 📥 How to Download

ClipSorter is available for both Windows and macOS. Follow the steps for your computer below.

### For Windows Users (Recommended)
1.  **Download:** Download the latest version of ClipSorter from the link provided by your administrator (usually a file named `ClipSorter-GUI-QuickStart.zip`).
2.  **Unzip:** Right-click the downloaded `.zip` file and select **"Extract All..."**. Choose a folder on your computer (like your Desktop).
3.  **Open:** Open the new folder and look for **`ClipSorter-GUI-App.exe`**.
4.  **Run:** Double-click the app to start!
    *   *Note: If Windows says "Windows protected your PC," click **More info** and then **Run anyway**.*

### For macOS Users
1.  **Download:** Download the **`clipSorter-macOS.zip`** file from the GitHub Actions tab (or the link provided).
2.  **Unzip:** Double-click the `.zip` file to reveal the **`ClipSorter.app`**.
3.  **Bypass Security:** On the first run, macOS will block the app because it's from an "unidentified developer."
    *   **Right-click** (or Control-click) the `ClipSorter.app` icon.
    *   Select **Open** from the menu.
    *   A message will appear; click **Open** again. You only need to do this once!

## 🚀 Getting Started (The 3-Step Sort)

ClipSorter is designed to be "non-destructive," meaning it **never** deletes or changes your original files. It always makes copies in a new folder.

### Step 1: Select your folder
Click the **"Select Source Folder"** button and choose the folder where your raw photos and videos are stored.

### Step 2: Choose where to save
ClipSorter will automatically suggest a new folder named `[YourFolder]_sorted`. You can change this if you want, but the default is usually best.

### Step 3: Start Sorting!
Click **"Start Sorting"**. 
*   A progress bar will show you how far along it is.
*   Once finished, a folder will automatically open with your organized results!

---

## 📂 Understanding Your Results

Inside your "Sorted" folder, you will find three main things:

1.  **`review/`**: This is where your best files are! They are organized into `photos`, `videos`, and `audio`.
2.  **`defects/`**: Files that failed quality checks (like very blurry photos or silent videos) are put here so they don't clutter your main gallery.
3.  **`_report.txt`**: A simple text file that explains exactly why each file was sorted into each bucket.

## ❓ Frequently Asked Questions

**Q: Why does the app take a minute to open on the first try?**  
A: The first time you run ClipSorter, it has to set up its "AI brain" (which detects subjects in photos). This is normal and will be much faster after the first time.

**Q: Will it delete my original files?**  
A: **No.** ClipSorter only reads your files and creates new, organized copies. Your originals remain exactly as they were.

**Q: Does it work with RAW camera files?**  
A: Yes! ClipSorter supports most professional camera formats (like `.ARW`, `.CR3`, `.MOV`, etc.) and converts them into standard files (`.jpg`, `.mp4`) for you.
